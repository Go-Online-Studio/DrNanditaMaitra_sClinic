# Static Public Assets Folder

Files placed in this `public` folder are served directly at the website's root path (`/`). They are not processed or compiled by Vite.

### Folder Contents:
* `favicon.ico` (Website browser tab icon, placed directly in this folder)
* `images/` (The main folder for all website images, logos, and gallery photos)
* `robots.txt` & `sitemap.xml`
